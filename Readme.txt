How To Run?

Open the folder where the directory is saved. If Python is installed on your machine, proceed further, else install it and then proceed.
1.    Open cmd and go the project folder.
2.    Run the command : python top10Searches.py <keyword>

What does this Script Do?

It accepts a command line argument as a keyword and searches it on the web using Google Search Engine. Top 10 search results are automatically opened in various tabs in your default browser.

Modules Used: sys, webbrowser
